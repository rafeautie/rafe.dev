# rafe.dev

My digital home. Part business card, part photo gallery, part playground.

Live at [rafe.dev](https://rafe.dev).

## What's here

- A landing page that greets you with a random photo from my collection
- A photography gallery
- A development page with projects I'm working on
- A little about me

## Built with

- [TanStack Start](https://tanstack.com/start) with React
- [Tailwind CSS](https://tailwindcss.com)
- [Cloudflare Workers](https://workers.cloudflare.com), with photos served from R2
